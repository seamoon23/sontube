# SonTube

SonTube는 보호자가 직접 등록한 YouTube 영상 안에서만 아이가 검색, 태그 필터, 시청할 수 있게 하는 아이 전용 영상서랍 웹/PWA MVP입니다.

## 실행 방법

1. `C:\codex\app\sontube` 폴더에서 의존성을 설치합니다.

```bash
npm install
```

2. 환경 파일을 만듭니다.

```bash
copy .env.example .env
```

3. SQLite DB와 Prisma Client를 준비합니다.

```bash
npm run db:push
npm run db:seed
```

4. 개발 서버를 실행합니다.

```bash
npm run dev
```

5. 브라우저에서 접근합니다.

- 관리자: `http://127.0.0.1:3000/admin`
- 영상 등록: `http://127.0.0.1:3000/admin/videos/new`
- 태그 관리: `http://127.0.0.1:3000/admin/tags`
- 아이 화면: `http://127.0.0.1:3000/kids`

## 구현된 기능

- 관리자 영상 등록, 수정, 목록
- YouTube URL에서 `videoId` 파싱
- 채널, 플레이리스트, 검색 URL 거부
- 중복 `videoId` 등록 방지
- 보호자 수동 입력 제목, 설명, 영상 시간 저장
- 안전 검토 상태와 공개 여부 관리
- 태그 생성, 수정, 활성/비활성, 사용 횟수 관리
- 최근 사용, 자주 사용, 제목/설명 키워드, 유사 영상 기반 추천 태그
- YouTube 기본 썸네일, 커스텀 썸네일 업로드, placeholder fallback
- 아이 화면의 승인/공개 영상만 표시
- 아이 화면 내부 제목/설명 검색, 태그 필터, 핫태그 표시
- 등록된 영상만 YouTube iframe으로 재생
- PWA manifest 기본 설정

## 미구현/후순위 기능

- 관리자 로그인과 세션 보호
- 서비스 워커 기반 설치 경험 고도화
- 시청 시간 제한
- 가족별 프로필 분리
- 태그 추천 규칙 편집 UI
- 커스텀 썸네일 삭제/교체 이력 관리
- 접근성 세부 점검과 모바일 실기기 테스트

## 테스트 방법

핵심 로직 테스트:

```bash
npm test
```

빌드 검증:

```bash
npm run build
```

수동 확인 경로:

- `관리자 > 영상 등록`: YouTube 영상 URL을 입력하고 `videoId`가 표시되는지 확인
- `관리자 > 영상 등록`: 채널 URL, 플레이리스트 URL, 검색 URL이 거부되는지 확인
- `관리자 > 태그 관리`: 태그 생성/수정 후 영상 등록 폼에 표시되는지 확인
- `아이 화면`: 보호자 확인 완료와 공개 상태인 영상만 보이는지 확인
- `아이 화면 > 영상`: iframe 재생 화면에 관리자 버튼이 없는지 확인
