# AGENTS.md - SonTube 작업 지침

## 프로젝트 목적

SonTube는 아이가 YouTube 전체를 검색하지 않고, 보호자가 직접 등록한 안전한 영상 목록 안에서만 검색, 태그 필터, 시청할 수 있게 하는 아이 전용 영상서랍 웹/PWA 프로젝트다.

이 앱은 특정 취향을 강요하는 도구가 아니다. 집 안에서 보는 영상만큼은 보호자가 직접 검토한 무해하고 좋은 영상들 안에서 아이가 마음 편히 고르게 하는 것이 목적이다.

## 절대 구현하지 말 것

- YouTube Data API 연동
- Google API Key 요구
- YouTube 전체 검색
- 댓글 조회, 표시, 분석
- 자막 조회, 다운로드, 분석
- 채널 영상 자동 가져오기
- 플레이리스트 영상 자동 가져오기
- 자체 영상 업로드
- AI 태깅 API 연동
- YouTube 광고 차단, 변조
- YouTube 다운로드, 오프라인 저장
- 백그라운드 재생

## 허용할 것

- YouTube URL에서 videoId 파싱
- YouTube iframe/embed로 등록된 영상 재생
- YouTube 기본 썸네일 URL 패턴 사용
- 보호자가 직접 입력한 제목, 설명, 영상 시간, 태그 저장
- 보호자가 직접 업로드한 커스텀 썸네일 저장
- 내부 DB에 저장된 승인 영상만 검색, 필터링

## 기술 스택

- Next.js App Router
- TypeScript
- SQLite
- Prisma
- Tailwind CSS
- Zod
- React Hook Form

## 작업 지침

- 아이 화면에는 관리자 기능, YouTube 전체 검색, 댓글, 채널 이동 UI를 노출하지 않는다.
- 외부 API 없이 내부 DB와 로컬 로직으로만 추천 태그를 만든다.
- 아이 검색은 보호자가 직접 작성한 제목과 검색 키워드 안에서만 수행한다.
- 아이 피드백/통계 기능은 직관적인 UX가 먼저이며, 복잡해지면 TODO에 보류 아이디어로 남긴다.
- 툴의 명령어나 메뉴를 사용하도록 가이드할 때는 접근 경로를 설명한다.
- 새 기능은 핵심 로직부터 테스트를 추가하고 검증한다.
